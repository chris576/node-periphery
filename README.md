# @msl5/node-periphery
GPIO and SPI libraries for the modern Node.js 24+ world.

## Installation

This package is published only on GitHub Packages.

1. Configure npm to use GitHub Packages for the `@msl5` scope:

```ini
@msl5:registry=https://npm.pkg.github.com
```

2. Authenticate against GitHub Packages (for private packages or rate-limited environments):

```ini
//npm.pkg.github.com/:_authToken=YOUR_GITHUB_TOKEN
```

3. Install the package:

```bash
npm install @msl5/node-periphery
```

## Breaking Change

Releases are no longer published to npmjs.com. The package name changed from `node-periphery` to `@msl5/node-periphery`.
